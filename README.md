# lainboard

An omega-simple and lightweight BBS system.

---

## Features

- Post posts
- Read posts
- Lain
- Cool looking

---

## Requirements

- PHP  
- SQLite3 on your server  
- Web server (Apache, Nginx, etc.)  
- Composer
- Twig for templates

---

## Installation

0. Install dependencies
1. Clone the repository into webserver root
2. Do composer install
3. Rename config.example.php to config.php
4. Configurate.
5. Connect to your webserver and post your first post to initialize the database.
6. ????????????
7. Profit have fun or smth
